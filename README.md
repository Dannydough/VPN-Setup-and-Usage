# VPN-Setup-and-Usage


**"(Provision a Virtual Machine in Azure)"**

1.) Navigate to https://whatismyipaddress.com/ from your local machine and record the IP address in a text file and create a Resource Group.

**"Create a Windows 10 Virtual Machine in Azure (Different Geographic Location)**


2.) Provision a Windows 10 Virtual Machine in a different geographic location (preferably a different country). Log into the Virtual Machine using Remote Desktop ---> Navigate to https://whatismyipaddress.com/ from within the VM and record the IP address in a text file.
